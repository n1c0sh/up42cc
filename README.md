# Coding challenge solution for Up42. 

### Instructions:
* [With Docker](#docker)
* [Without Docker](#maven-wrapper)

# Docker
1) Execute: `docker-compose up`

# Maven wrapper
1) Install JDK 11
2) Make sure JAVA_HOME environment variable is pointing to Jdk11 installation folder
2) Go to project folder
3) Execute: 
    * For Linux or Unix: `mvnw spring-boot:run`
    * For Windows: `mvnw.cmd spring-boot:run`
4) Proceed to [http://localhost:8080/features](http://localhost:8080/features)
